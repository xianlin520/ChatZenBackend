## 后端接口
### 应用技术
+ 使用**SpringBoot3**框架作为后端接口框架
+ **MySQL8**作为用户数据储存数据库
+ **Redis**作为缓存数据库
+ 预计添加**图床**和**资源数据库**, 用于存储图片文件类型数据
+ 计划使用**Spring Security**作为鉴权框架
+ 使用**RabbitMQ**消息队列做中间件
+ 使用**Logback**作为后端日志系统, 并接入日志数据库
+ 计划加入**微服务**负载均衡等技术, 以练习为目的
+ 计划使用**RESTful Api**和 **WebSocket **等交互方式来进行数据传输
+ 可能引入**接口加密**, 具体选型未定
